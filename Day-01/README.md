# DevConf 2026 Conference Website

This project is a multi-page HTML5 website created as part of an academic assignment.

## Pages Included
- index.html (Home)
- about.html (About the Conference)
- speakers.html (Keynote Speakers)
- schedule.html (Conference Schedule)
- sponsors.html (Sponsors)

## Technologies Used
- HTML5 only

## Features
- Semantic HTML structure
- Consistent navigation menu
- Tables, lists, and sections
- Speaker profiles and sponsor tiers
- Fully responsive meta setup
