# Day 10 – Brand Login & Profile Clone

## 📌 Project Overview
This project replicates a modern authentication interface inspired by popular platforms like Instagram or X (Twitter).  
It focuses on building a clean and responsive form UI while implementing HTML validation and interactive CSS effects.

---

## 🎯 Learning Objectives

- Replicate professional UI layouts with precise spacing and alignment
- Implement HTML5 form validation
- Build responsive layouts using pure CSS
- Use pseudo-classes for interactive effects like hover and focus

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Responsive Design
- CSS Grid & Flexbox
