# Day 12 – Portfolio Performance Audit

Performed a technical audit of a DevConf website and optimized its structure for accessibility and performance.

## Improvements

- Added semantic HTML structure
- Ensured all images include meaningful alt text
- Fixed heading hierarchy
- Added ARIA labels for accessibility
- Refactored CSS using root variables
- Optimized animations using transform and opacity
- Implemented responsive layout with CSS Grid
